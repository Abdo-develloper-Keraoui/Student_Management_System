JAVA QUICK REFERENCE CHEAT SHEET

--- PRIMITIVE TYPES ---
Type        Size    Range/Values
---------------------------------------------------------
byte        1 byte  -128 to 127
short       2 bytes -32,768 to 32,767
int         4 bytes -2^31 to 2^31-1 (-2,147,483,648 to 2,147,483,647)
long        8 bytes -2^63 to 2^63-1 (add L suffix to literals: 100L)
float       4 bytes ~6-7 decimal digits (add F suffix: 3.14F)
double      8 bytes ~15 decimal digits (default for decimal values)
char        2 bytes '\u0000' to '\uffff' (single Unicode character)
boolean     1 bit   true or false

--- COMMON REFERENCE TYPES (CLI projects) ---
1. String (java.lang)
   - Immutable character sequence
   - Common methods: length(), charAt(), substring(), equals(),
     contains(), toLowerCase(), toUpperCase(), split()

2. ArrayList<E> (java.util)
   - Resizable array implementation
   - Common methods: add(), get(), set(), remove(), size(),
     contains(), indexOf(), isEmpty()

3. HashMap<K,V> (java.util)
   - Key-value pair storage
   - Common methods: put(), get(), containsKey(), containsValue(),
     remove(), keySet(), values(), size()

4. LocalDate/LocalTime/LocalDateTime (java.time)
   - Date and time manipulation (ISO-8601)
   - Common methods: now(), of(), parse(), plusDays(), minusMonths(),
     getYear(), isAfter(), isBefore()

5. Scanner (java.util)
   - Text parser for input streams
   - Common methods: next(), nextInt(), nextLine(), hasNext(),
     useDelimiter(), close()

6. LinkedList<E> (java.util)
   - Doubly-linked list implementation
   - Common methods: addFirst(), addLast(), removeFirst(),
     removeLast(), peek(), poll()

7. HashSet<E> (java.util)
   - Unordered collection of unique elements
   - Common methods: add(), contains(), remove(), size(), isEmpty()

8. PriorityQueue<E> (java.util)
   - Priority-based queue
   - Common methods: add(), offer(), poll(), peek(), size()

9. Optional<T> (java.util)
   - Container for nullable objects
   - Common methods: of(), ofNullable(), isPresent(), get(),
     orElse(), ifPresent()

--- NOTES ---
• Wrapper classes for primitives:
  Integer, Double, Boolean, Character, etc.
• Use equals() instead of == for object comparison
• Autoboxing: Automatic conversion between primitives and wrappers
• Common packages to import:
  java.util.*, java.time.*, java.io.*
• Always close resources (Files, Scanners) with try-with-resources
• Use StringBuilder for mutable string operations
• Remember to handle exceptions (IOException, NullPointerException, etc.)
